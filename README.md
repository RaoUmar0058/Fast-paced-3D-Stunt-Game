🛠️ Technical Highlights
Physics Engine: Built using Cannon.js for realistic collision and movement.
3D Rendering: Powered by Three.js for high-performance graphics.
Responsive: Optimized for smooth gameplay in modern browsers.
Procedural: Dynamic track elements keep every run fresh and challenging.
⌨️ Controls
W: Accelerate
A / D: Steer / Balance for Stunts
Space: Handbrake (Customizable)
